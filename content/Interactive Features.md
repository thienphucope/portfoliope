# Interactive Features & Detective Tools

## 🇬🇧 English: System Capabilities
The **Case Archives** project integrates several custom-built tools designed to provide an immersive and functional environment for both investigative storytelling and technical demonstration.

### 1. AI Chat Vault
A state-of-the-art conversational interface powered by streaming protocols.
- **Context Awareness:** The AI is tuned to the "detective" persona, capable of discussing project details and case lore.
- **Real-time Streaming:** Messages are delivered token-by-token for a fluid, responsive experience.
- **Bespoke Integration:** Features a custom UI that blends perfectly with the vault's aesthetic.

### 2. Dynamic Block Editor
A custom-engineered Markdown editor optimized for non-linear documentation.
- **Block-Based Logic:** Text is managed in discrete blocks, allowing for easier manipulation and rendering.
- **Live Preview:** Immediate visualization of Markdown syntax, including tables, code blocks, and math equations.
- **Local Persistence:** Utilizes browser storage to cache drafts, preventing data loss during sessions.

### 3. Recursive File Tree
A powerful navigation system mimicking a professional file manager.
- **Nested Structures:** Supports infinite levels of folders and files.
- **Direct Interaction:** Integration with the main accordion UI to open records with a single click.
- **Search Functionality:** Built-in filtering to locate specific investigative notes quickly.

### 4. Atmospheric Multimedia
Synchronized audio and video systems to enhance the "Detective's Office" experience.
- **Background Player:** A customized YouTube integration that provides atmospheric visuals without interrupting the workflow.
- **Positional Audio:** Ambient soundscapes that change based on user interaction.

## 🇻🇳 Tiếng Việt: Các Tính năng Hệ thống
Dự án **Kho Lưu Trữ Hồ Sơ** tích hợp nhiều công cụ được xây dựng tùy chỉnh nhằm cung cấp một môi trường nhập vai và đầy đủ chức năng cho cả việc kể chuyện trinh thám lẫn trình diễn kỹ thuật.

### 1. AI Chat Vault
Giao diện hội thoại hiện đại được hỗ trợ bởi các giao thức truyền phát dữ liệu (streaming).
- **Nhận diện Ngữ cảnh:** AI được tinh chỉnh theo phong cách "thám tử", có khả năng thảo luận về chi tiết dự án và cốt truyện của các vụ án.
- **Truyền phát Thời gian thực:** Tin nhắn được gửi đi từng phần (token-by-token) để tạo trải nghiệm mượt mà và phản hồi nhanh chóng.
- **Tích hợp Tùy biến:** Giao diện người dùng tùy chỉnh hoàn toàn phù hợp với thẩm mỹ của kho lưu trữ.

### 2. Trình chỉnh sửa Khối (Block Editor)
Một trình chỉnh sửa Markdown được thiết kế riêng, tối ưu hóa cho việc lập tài liệu phi tuyến tính.
- **Logic Dựa trên Khối:** Văn bản được quản lý trong các khối riêng biệt, cho phép thao tác và render dễ dàng hơn.
- **Xem trước Trực tiếp:** Hiển thị tức thì các cú pháp Markdown, bao gồm bảng, khối mã và các phương trình toán học.
- **Lưu trữ Cục bộ:** Sử dụng bộ nhớ trình duyệt để lưu tạm bản nháp, ngăn chặn mất dữ liệu trong phiên làm việc.

### 3. Cây Thư mục Đệ quy
Hệ thống điều hướng mạnh mẽ mô phỏng một trình quản lý tệp chuyên nghiệp.
- **Cấu trúc Phân cấp:** Hỗ trợ vô số cấp độ thư mục và tệp tin.
- **Tương tác Trực tiếp:** Kết nối với giao diện Accordion chính để mở hồ sơ chỉ với một cú nhấp chuột.
- **Chức năng Tìm kiếm:** Bộ lọc tích hợp giúp xác định vị trí các ghi chú điều tra cụ thể một cách nhanh chóng.

### 4. Đa phương tiện Nhập vai
Hệ thống âm thanh và hình ảnh đồng bộ nhằm tăng cường trải nghiệm "Văn phòng Thám tử".
- **Trình phát Nền:** Tích hợp YouTube tùy chỉnh cung cấp hình ảnh không khí mà không làm gián đoạn luồng công việc.
- **Âm thanh Không gian:** Các dải âm thanh môi trường thay đổi dựa trên tương tác của người dùng.

## 🇨🇳 中文 (Chinese): 系统功能与侦探工具
**“案件档案库”**项目集成了多个定制开发的工具，旨在为侦探叙事和技术展示提供一个沉浸式且功能完善的环境。

### 1. AI 聊天保险库 (AI Chat Vault)
一个由流协议驱动的先进对话界面。
- **上下文感知：** AI 针对“侦探”身份进行了微调，能够讨论项目细节和案件背景。
- **实时流传输：** 消息以 token 为单位逐字传输，提供流畅且响应迅速的体验。
- **定制化集成：** 采用定制 UI，与保险库的美学风格完美融合。

### 2. 动态块编辑器 (Dynamic Block Editor)
专为非线性文档优化的定制 Markdown 编辑器。
- **块逻辑：** 文本以离散块的形式进行管理，便于操作和渲染。
- **实时预览：** Markdown 语法的即时可视化，包括表格、代码块和数学公式。
- **本地持久化：** 利用浏览器存储缓存草稿，防止会话期间数据丢失。

### 3. 递归文件树
模拟专业文件管理器的强大导航系统。
- **嵌套结构：** 支持无限层级的文件夹和文件。
- **直接交互：** 与主折叠面板 (Accordion) UI 集成，点击即可打开记录。
- **搜索功能：** 内置过滤功能，可快速定位特定的调查笔记。

### 4. 沉浸式多媒体
同步的视听系统，增强“侦探办公室”的沉浸感。
- **背景播放器：** 定制的 YouTube 集成，提供氛围影像且不干扰工作流程。
- **环境音效：** 根据用户交互而变化的背景环境音。
