## OpenAI
Ok, đây là list nguồn tin "ngay tại nguồn" để cậu hóng biến và cập nhật kỹ thuật về **OpenAI**:

1. [https://openai.com/news/](https://openai.com/news/) - Blog chính chủ, nơi release các research paper, model mới và thông báo quan trọng nhất.
2. [https://x.com/OpenAI](https://x.com/OpenAI) - Kênh X (Twitter) chính thức, cập nhật news nhanh gọn, retweet các demo xịn từ cộng đồng.
3. [https://x.com/sama](https://x.com/sama) - Twitter của Sam Altman (CEO), chuyên đăng tweet ẩn ý, hóng drama nội bộ hay định hướng tương lai là phải xem ở đây.
4. [https://x.com/gdb](https://x.com/gdb) - Twitter của Greg Brockman (Co-founder), hay post demo kỹ thuật thực tế, show hàng tính năng mới.
5. [https://community.openai.com/](https://community.openai.com/) - Forum cho Developer, vào đây để xem anh em dev quốc tế kêu ca về API limit, lỗi model hoặc trick prompt hay.
6. [https://www.reddit.com/r/OpenAI/](https://www.reddit.com/r/OpenAI/) - Subreddit lớn nhất, hỗn hợp giữa tin tức, tin đồn (rumors), meme và user feedback.
7. [https://github.com/openai/openai-cookbook](https://github.com/openai/openai-cookbook) - Repo GitHub chứa "công thức nấu ăn" (code mẫu), best practice để tích hợp API OpenAI.
8. [https://www.youtube.com/@OpenAI](https://www.youtube.com/@OpenAI) - Kênh YouTube để xem demo trực quan (Sora, Voice mode) và sự kiện DevDay.
9. [https://status.openai.com/](https://status.openai.com/) - Trang check uptime, quan trọng cho dev để biết khi nào API sập hay lag.
10. [https://news.ycombinator.com/from?site=openai.com](https://www.google.com/search?q=https://news.ycombinator.com/from%3Fsite%3Dopenai.com) - Hacker News (lọc domain openai), nơi các dev lão làng thảo luận sâu về tech và tác động ngành.

## Grok 
List nguồn để cậu theo dõi **xAI** (Grok), hệ sinh thái này đặc biệt vì news chủ yếu nổ ra trên platform X (Twitter) đầu tiên:

1. [https://x.ai/blog](https://x.ai/blog) - Trang chủ, nơi công bố chính thức về model architecture và các bài research nặng đô.
2. [https://x.com/xai](https://x.com/xai) - Kênh X chính chủ, chắc chắn là nơi cập nhật nhanh nhất mọi thứ về công ty.
3. [https://x.com/elonmusk](https://x.com/elonmusk) - "Tổng quản" Elon, follow ông này để biết roadmap (dù time hay delay) và leak tính năng mới.
4. [https://x.com/grok](https://x.com/grok) - Account tập trung vào sản phẩm chatbot Grok, các update về visual, image generation.
5. [https://console.x.ai/](https://console.x.ai/) - Khu vực cho Dev: lấy API key, xem pricing, và playground để test model.
6. [https://docs.x.ai/](https://docs.x.ai/) - Tài liệu kỹ thuật chi tiết, hướng dẫn function calling và SDK cho anh em code.
7. [https://github.com/xai-org](https://github.com/xai-org) - GitHub repo, nơi họ đã từng open-source Grok-1 weights và các tool liên quan.
8. [https://www.reddit.com/r/xAI/](https://www.reddit.com/r/xAI/) - Subreddit thảo luận chung về công ty, hạ tầng siêu máy tính Colossus.
9. [https://x.com/i/lists/1720667073286971527](https://www.google.com/search?q=https://x.com/i/lists/1720667073286971527) - (Unofficial List) Một list trên X tổng hợp các kỹ sư của xAI, follow để xem họ đang cook món gì.

## Gemini
Ok, đây là bộ link để theo dõi **Google Gemini** (DeepMind & Google AI), hệ sinh thái này khá rộng nên tôi lọc ra những chỗ "nhiều thịt" nhất cho dev:

1. [https://deepmind.google/discover/blog/](https://deepmind.google/discover/blog/) - "Thánh địa" research, nơi công bố các paper về Gemini, AlphaFold và kỹ thuật nền tảng.
2. [https://blog.google/technology/ai/](https://blog.google/technology/ai/) - Kênh tin tức official về sản phẩm thương mại, feature mới trên Android/Workspace.
3. [https://ai.google.dev/](https://ai.google.dev/) - Cổng thông tin chính cho Dev: Docs, API references, và các tools mới nhất.
4. [https://aistudio.google.com/](https://aistudio.google.com/) - Google AI Studio, nơi test model (thường free), chỉnh parameters nhanh nhất cho dev.
5. [https://x.com/GoogleDeepMind](https://x.com/GoogleDeepMind) - Kênh X cập nhật nhanh về research mới, thường có visual/video giải thích dễ hiểu.
6. [https://x.com/demishassabis](https://x.com/demishassabis) - CEO DeepMind, người nắm giữ "long mạch" và lộ trình phát triển của Gemini.
7. [https://x.com/JeffDean](https://x.com/JeffDean) - Chief Scientist của Google, follow ông này để hóng về kiến trúc hệ thống và update cốt lõi.
8. [https://github.com/google-gemini/cookbook](https://github.com/google-gemini/cookbook) - Repo chứa code mẫu, prompt guide chất lượng cao để tích hợp API.
9. [https://www.reddit.com/r/GeminiAI/](https://www.reddit.com/r/GeminiAI/) - Cộng đồng Reddit thảo luận về prompt, so sánh benchmark và "kêu ca" về filter/censorship.
10. [https://www.youtube.com/@GoogleCloudTech](https://www.youtube.com/@GoogleCloudTech) - Kênh YouTube dạy cách deploy Gemini trên Vertex AI, hợp cho ai làm enterprise.
