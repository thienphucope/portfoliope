# Development Journey - Portfolio Project

## 🇬🇧 English: Technical Milestones
The development of this project has been a journey of exploring the boundaries between web applications and interactive storytelling.

### 1. The Foundation (Alpha)
- **Project Initiation:** Setup Next.js 15 with Turbopack for high-speed development.
- **Accordion Core:** Engineered the horizontal accordion UI logic, managing complex state transitions and liquid-smooth scrolling animations.

### 2. The Vault System (Beta)
- **File Management:** Implemented a recursive File Tree and a robust Markdown rendering engine.
- **Interactive Editing:** Built the **Block Editor**, allowing real-time record updates with local caching and server-side persistence.
- **AI Integration:** Integrated the **AI Chat Vault**, featuring streaming responses and context-awareness within the detective theme.

### 3. Polish & Optimization (V1.0)
- **Hybrid Architecture:** Refactored main pages into Server and Client components to optimize for AdSense crawlers and initial load speed.
- **Global Reach:** Implemented multi-language documentation (EN/VN/CN).
- **Responsive Design:** Fine-tuned the vertical accordion behavior for mobile devices and optimized multimedia performance.

## 🇻🇳 Tiếng Việt: Các Cột mốc Kỹ thuật
Quá trình phát triển dự án này là một hành trình khám phá ranh giới giữa ứng dụng web và nghệ thuật kể chuyện tương tác.

### 1. Nền móng (Alpha)
- **Khởi tạo dự án:** Thiết lập Next.js 15 với Turbopack để tối ưu tốc độ phát triển.
- **Cốt lõi Accordion:** Xây dựng logic cho giao diện Accordion nằm ngang, xử lý các trạng thái chuyển đổi phức tạp và hiệu ứng cuộn mượt mà.

### 2. Hệ thống Vault (Beta)
- **Quản lý tệp:** Triển khai Cây thư mục (File Tree) đệ quy và công cụ render Markdown mạnh mẽ.
- **Chỉnh sửa tương tác:** Xây dựng **Block Editor**, cho phép cập nhật hồ sơ trong thời gian thực với bộ nhớ đệm cục bộ và lưu trữ phía máy chủ.
- **Tích hợp AI:** Tích hợp **AI Chat Vault**, hỗ trợ phản hồi dạng truyền phát (streaming) và nhận diện ngữ cảnh theo chủ đề thám tử.

### 3. Hoàn thiện & Tối ưu hóa (V1.0)
- **Kiến trúc Hybrid:** Tái cấu trúc các trang chính thành Server và Client component để tối ưu cho crawler của AdSense và tốc độ tải trang ban đầu.
- **Tiếp cận Toàn cầu:** Triển khai tài liệu đa ngôn ngữ (Anh/Việt/Trung).
- **Thiết kế Đáp ứng:** Tinh chỉnh hành vi Accordion dọc cho thiết bị di động và tối ưu hóa hiệu suất đa phương tiện.

## 🇨🇳 中文 (Chinese): 技术里程碑
本项目的开发是一段探索 Web 应用与互动叙事边界的旅程。

### 1. 基础阶段 (Alpha)
- **项目初始化：** 使用 Next.js 15 和 Turbopack 搭建环境，以实现高速开发。
- **折叠面板核心：** 开发了水平折叠面板 (Accordion) UI 逻辑，管理复杂的页面状态切换和丝滑的滚动动画。

### 2. 保险库系统 (Beta)
- **文件管理：** 实现了递归文件树和强大的 Markdown 渲染引擎。
- **交互式编辑：** 构建了**块编辑器 (Block Editor)**，支持具有本地缓存和服务器端持久化的实时记录更新。
- **AI 集成：** 集成了 **AI 聊天保险库 (AI Chat Vault)**，支持流式响应并结合侦探主题的上下文感知。

### 3. 完善与优化 (V1.0)
- **混合架构：** 将主要页面重构为服务器 (Server) 和客户端 (Client) 组件，以优化 AdSense 爬虫抓取和首屏加载速度。
- **全球化支持：** 实现了多语言文档（英/越/中）。
- **响应式设计：** 为移动设备微调了纵向折叠面板行为，并优化了多媒体加载性能。
